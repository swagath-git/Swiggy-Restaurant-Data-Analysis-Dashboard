# Swiggy-Restaurant-Data-Analysis-Dashboard
Developed a Power BI dashboard to analyze Swiggy restaurant data and uncover insights on cuisine trends, pricing, delivery time and high-demand areas.

##  Project Overview
This project focuses on analyzing Swiggy restaurant data to understand food category distribution, restaurant density, pricing variations and delivery time across different cities and areas. The insights are visualized using an interactive Power BI dashboard.

##  Objective
To provide data-driven insights that help improve customer experience, pricing strategy, delivery efficiency and business expansion planning.

##  Dataset Description
The dataset contains restaurant-level details including:
- City
- Area
- Restaurant ID
- Food Type
- Average Rating
- Average Price
- Delivery Time

##  Tools & Technologies
- **Power BI** – Dashboard design and visualization
- **Power Query** – Data cleaning and transformation
- **DAX** – KPI calculations
- **Excel / CSV** – Dataset

##  Dashboard Highlights
- **KPIs:** Avg Rating, Avg Price, Total Restaurants, Avg Delivery Time
- **Top 10 Areas with Highest Restaurant Count**
- **Top 10 Most Served Food Types**
- **Top 10 Cities with Maximum Restaurants**
- **Most Expensive Areas by Avg Price**
- Dynamic filters for **City**, **Area**, and **Food Type**

- Expand in **high-demand areas with increasing order frequency**.

- Reduce delivery time variance in slower regions to improve customer satisfaction.
